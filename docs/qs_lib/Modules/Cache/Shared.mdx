# Shared

A structure holding cached results of functions, often for constants or rarely updated data.

## Adding New Cached Values

You can cache values indefinitely or set them to expire after a specified duration.

```lua
cache(identifier, callback, duration)
```

- identifier: `string`
  - A unique label to store and retrieve the cached data.
- callback: `function`
  - A function executed when the cache is empty or expired.
- duration?: `number`
  - Time in milliseconds after which the cache expires.

```lua
while true do
    Wait(2000)
    
    print(cache('playerHealth', function() 
        return GetPlayerHealth() 
    end, 10000))
end

-- output:
> 100
> 100
> 100
> 100
> 100
> 95
> 95
```

## Default Values

The following cached entries are available:

- resource: `string`
  - Result of [GetCurrentResourceName](https://docs.fivem.net/natives/?_0xE5E9EBBB).
- game: `'fxserver' | 'fivem' | 'redm'`
  - Result of [GetGameName](https://docs.fivem.net/natives/?_0xE8EAA18B).
