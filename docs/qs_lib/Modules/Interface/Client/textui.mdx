# TextUI

## lib.showTextUI

Displays the TextUI window on the screen.

<Alert variant="warning" className="mb-4">
  <AlertTitle>Warning</AlertTitle>
  <AlertDescription>
    Do not call this function every frame; it is meant to be used as a toggle.
  </AlertDescription>
</Alert>

<Tabs defaultValue="lua" className="mt-4">
  <TabsList className="grid w-full grid-cols-2">
    <TabsTrigger value="lua">Lua</TabsTrigger>
    <TabsTrigger value="js">JavaScript</TabsTrigger>
  </TabsList>
  <TabsContent value="lua">
    ```lua
    lib.showTextUI(message, config)
    ```
  </TabsContent>
  <TabsContent value="js">
    ```js
    import lib from '@emanueldev1/qs_lib/client';

    lib.showTextUI(message, config);
    ```
  </TabsContent>
</Tabs>

- message: `string`
- config?: `table` (`object`)
  - position?: `'right-center'` | `'left-center'` | `'top-center'` | `'bottom-center'`
    - Default: `'right-center'`
  - icon?: `string` | `table` (`array`)
  - iconColor?: `string`
  - iconAnimation?: `'spin'` | `'spinPulse'` | `'spinReverse'` | `'pulse'` | `'beat'` | `'fade'` | `'beatFade'` | `'bounce'` | `'shake'`
  - style?: React.CSSProperties
  - alignIcon?: `'top'` | `'center'`
    - Default: `'center'`

## lib.isTextUIOpen

Checks if the TextUI is currently visible, returning the displayed text as a second value.

<Tabs defaultValue="lua" className="mt-4">
  <TabsList className="grid w-full grid-cols-2">
    <TabsTrigger value="lua">Lua</TabsTrigger>
    <TabsTrigger value="js">JavaScript</TabsTrigger>
  </TabsList>
  <TabsContent value="lua">
    ```lua
    local isVisible, currentText = lib.isTextUIOpen()
    ```
  </TabsContent>
  <TabsContent value="js">
    ```js
    import lib from '@emanueldev1/qs_lib/client';

    const [isVisible, currentText] = lib.isTextUIOpen();
    ```
  </TabsContent>
</Tabs>
