# Skill Check

## lib.skillCheck

Initiates a skill check with a specified difficulty level.

<Tabs defaultValue="lua" className="mt-4">
  <TabsList className="grid w-full grid-cols-2">
    <TabsTrigger value="lua">Lua</TabsTrigger>
    <TabsTrigger value="js">JavaScript</TabsTrigger>
  </TabsList>
  <TabsContent value="lua">
    ```lua
    lib.skillCheck(levels, keys)
    ```
  </TabsContent>
  <TabsContent value="js">
    ```js
    import lib from '@emanueldev1/qs_lib/client';

    lib.skillCheck(levels, keys);
    ```
  </TabsContent>
</Tabs>

- levels: `'easy'` | `'medium'` | `'hard'` | `table`
  - Predefined difficulty levels:
    - `'easy'`: `{ areaSize: 50, speedMultiplier: 1 }`
    - `'medium'`: `{ areaSize: 40, speedMultiplier: 1.5 }`
    - `'hard'`: `{ areaSize: 25, speedMultiplier: 1.75 }`
  - Custom difficulty can be defined using a table:
    - areaSize: `number`
      - Size of the success zone in degrees.
    - speedMultiplier: `number`
      - Speed multiplier for the indicator.
- keys?: `string[]`
  - Randomly selects a key from the provided list for each skill check.
  - Defaults to `'e'` if not specified.

## Examples

### Example 1: Lockpicking Challenge

Runs a skill check sequence for lockpicking with increasing difficulty.

<Tabs defaultValue="lua" className="mt-4">
  <TabsList className="grid w-full grid-cols-2">
    <TabsTrigger value="lua">Lua</TabsTrigger>
    <TabsTrigger value="js">JavaScript</TabsTrigger>
  </TabsList>
  <TabsContent value="lua">
    ```lua
    local success = lib.skillCheck({'easy', 'medium', 'hard'}, {'q', 'e', 'r'})

    if success then
        print("Lockpicking successful!")
    else
        print("Lockpicking failed.")
    end
    ```
  </TabsContent>
  <TabsContent value="js">
    ```js
    import lib from '@emanueldev1/qs_lib/client';

    const success = await lib.skillCheck(['easy', 'medium', 'hard'], ['q', 'e', 'r']);

    if (success) {
      console.log("Lockpicking successful!");
    } else {
      console.log("Lockpicking failed.");
    }
    ```
  </TabsContent>
</Tabs>
