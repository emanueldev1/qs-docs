# Server

## lib.versionCheck

Checks the resource version against the latest release on GitHub using its release API.

```lua
lib.versionCheck(repoName)
```

- repoName: `string`

### Examples

#### Example 1: Checking Version for a Core Resource

Verifies the version of a core resource against the latest release on GitHub.

  <TabsList className="grid w-full grid-cols-2">

    <TabsTrigger value="js">JavaScript</TabsTrigger>
  </TabsList>
  <TabsContent value="lua">
    ```lua
    lib.versionCheck('emanueldev1/qs_lib')
    print("Version check initiated for qs_lib.")
    ```
  </TabsContent>
  <TabsContent value="js">

  </TabsContent>
